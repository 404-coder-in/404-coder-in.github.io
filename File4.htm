📘 Class 11 Mathematics Formulas

Chapter 1: Sets

1. Set Representation

▸ Roster form: A = {1, 2, 3}
▸ Set-builder form: A = {x: x ∈ ℕ, x < 4}

2. Types of Sets

▸ Empty, Finite, Infinite, Equal, Subset, Power Set, Universal Set

3. Venn Diagrams

▸ Pictorial representation of sets

4. Operations on Sets

▸ Union: A ∪ B = {x: x ∈ A or x ∈ B}
▸ Intersection: A ∩ B = {x: x ∈ A and x ∈ B}
▸ Difference: A – B = {x: x ∈ A and x ∉ B}
▸ Complement: A′ = {x: x ∉ A}

5. Important Properties

▸ n(A ∪ B) = n(A) + n(B) – n(A ∩ B)
▸ De Morgan’s Laws:
    (A ∪ B)′ = A′ ∩ B′
    (A ∩ B)′ = A′ ∪ B′

Chapter 2: Relations and Functions

1. Cartesian Product

▸ A × B = {(a, b): a ∈ A, b ∈ B}
▸ If A has m elements, B has n → A × B has mn elements

2. Relation

▸ A relation from A to B is a subset of A × B
▸ Types: Empty, Universal, Identity

3. Function

▸ A relation is a function if every element of A has a unique image in B
▸ Domain: Set of all inputs
▸ Co-domain: Set where outputs lie
▸ Range: Actual outputs from domain

4. Types of Functions

▸ One-one (Injective), Onto (Surjective), Bijective
▸ Constant, Identity, Polynomial, Rational, Modulus

Chapter 3: Trigonometric Functions

1. Basic Trigonometric Ratios

▸ sin θ = Perpendicular / Hypotenuse
▸ cos θ = Base / Hypotenuse
▸ tan θ = Perpendicular / Base = sin θ / cos θ
▸ cot θ = 1 / tan θ = cos θ / sin θ
▸ sec θ = 1 / cos θ
▸ cosec θ = 1 / sin θ

2. Fundamental Identities

▸ sin²θ + cos²θ = 1
▸ 1 + tan²θ = sec²θ
▸ 1 + cot²θ = cosec²θ

3. Trigonometric Values (0° to 90°)

▸ sin 0° = 0, sin 30° = 1/2, sin 45° = 1/√2, sin 60° = √3/2, sin 90° = 1
▸ cos 0° = 1, cos 30° = √3/2, cos 45° = 1/√2, cos 60° = 1/2, cos 90° = 0
▸ tan 0° = 0, tan 30° = 1/√3, tan 45° = 1, tan 60° = √3, tan 90° = ∞

4. Allied Angle Identities

▸ sin(180° – θ) = sin θ
▸ cos(180° – θ) = –cos θ
▸ tan(180° – θ) = –tan θ
▸ sin(90° ± θ) = cos θ
▸ cos(90° ± θ) = sin θ
▸ tan(90° ± θ) = ±cot θ

5. Angle Conversion (Degrees ↔ Radians)

▸ 180° = π radians
▸ 1 radian = 180° / π
▸ 1° = π / 180 radians

6. Periodicity and Signs (ASTC Rule)

▸ sin(θ + 2π) = sin θ
▸ cos(θ + 2π) = cos θ
▸ tan(θ + π) = tan θ
▸ cot(θ + π) = cot θ

7. Negative Angle Identities

▸ sin(–θ) = –sin θ
▸ cos(–θ) = cos θ
▸ tan(–θ) = –tan θ
▸ cosec(–θ) = –cosec θ
▸ sec(–θ) = sec θ
▸ cot(–θ) = –cot θ

8. Sum and Difference Formulas

▸ sin(A ± B) = sin A cos B ± cos A sin B
▸ cos(A ± B) = cos A cos B ∓ sin A sin B
▸ tan(A ± B) = (tan A ± tan B) / (1 ∓ tan A tan B)

9. Double Angle Formulas

▸ sin 2A = 2 sin A cos A
▸ cos 2A = cos²A – sin²A = 2cos²A – 1 = 1 – 2sin²A
▸ tan 2A = (2 tan A) / (1 – tan²A)

10. Product to Sum Formulas

▸ sin A sin B = ½[cos(A–B) – cos(A+B)]
▸ cos A cos B = ½[cos(A–B) + cos(A+B)]
▸ sin A cos B = ½[sin(A+B) + sin(A–B)]

11. Sum to Product Formulas

▸ sin A + sin B = 2 sin[(A+B)/2] cos[(A–B)/2]
▸ sin A – sin B = 2 cos[(A+B)/2] sin[(A–B)/2]
▸ cos A + cos B = 2 cos[(A+B)/2] cos[(A–B)/2]
▸ cos A – cos B = –2 sin[(A+B)/2] sin[(A–B)/2]

Chapter 4: Complex Numbers and Quadratic Equations

1. Definition of Complex Numbers

▸ A complex number is of the form z = x + iy, where x, y ∈ ℝ and i = √–1
▸ x is called the real part: Re(z) = x
▸ y is called the imaginary part: Im(z) = y

2. Algebra of Complex Numbers

▸ (a + ib) + (c + id) = (a + c) + i(b + d)
▸ (a + ib)(c + id) = (ac – bd) + i(ad + bc)
▸ (a + ib) / (c + id) = [(ac + bd) + i(bc – ad)] / (c² + d²)

3. Modulus and Conjugate

▸ Modulus of z = x + iy is |z| = √(x² + y²)
▸ Conjugate of z = x + iy is z̅ = x – iy
▸ z × z̅ = |z|² = x² + y²

4. Argand Plane Representation

▸ Complex number z = x + iy is represented as point (x, y) on Argand Plane
▸ Distance from origin = |z| = √(x² + y²)
▸ Argument of z: arg(z) = tan⁻¹(y/x)

5. Quadratic Equations

▸ Standard form: ax² + bx + c = 0
▸ Discriminant: D = b² – 4ac
▸ Nature of roots:
    ▸ D > 0 → Real and distinct
    ▸ D = 0 → Real and equal
    ▸ D < 0 → Complex and conjugate pair
▸ Roots formula: x = [–b ± √D] / (2a)

6. Properties of Roots

▸ Sum of roots = –b/a
▸ Product of roots = c/a
▸ If α and β are roots: then equation is a(x – α)(x – β) = 0

7. Nature of Roots with Complex Numbers

▸ If D < 0, roots are complex and conjugate: α = p + iq, β = p – iq
▸ Graph of such quadratic never touches x-axis

8. Important Identities

▸ i² = –1
▸ i³ = –i
▸ i⁴ = 1
▸ iⁿ = i^(n mod 4)

Chapter 5: Linear Inequalities

1. Linear Inequality Definition

▸ An expression of the form ax + b < c or ax + b > c, where a, b, c ∈ ℝ and a ≠ 0
▸ Solution set: values of variable satisfying the inequality

2. Rules for Solving Inequalities

▸ Same number can be added/subtracted on both sides
▸ Can multiply/divide both sides by same positive number
▸ If multiplied/divided by negative number, inequality sign reverses

3. Graphical Representation

▸ Represented on number line
▸ Open circle ⭕ for < or >
▸ Filled circle ⚫ for ≤ or ≥

4. Linear Inequalities in Two Variables

▸ General form: ax + by ≤ c, ax + by ≥ c
▸ Solution: Region in coordinate plane satisfying inequality
▸ Boundary line: ax + by = c (dashed or solid)
▸ Use test point (e.g., (0,0)) to decide solution region

5. System of Inequalities

▸ Solving multiple inequalities together
▸ Graphical solution: shaded region common to all inequalities
▸ Feasible region: all points satisfying all conditions

Chapter 6: Permutations and Combinations

1. Fundamental Principle of Counting

▸ If an event can occur in m ways and another event in n ways, then both can occur in m × n ways

2. Factorial

▸ n! = n × (n–1) × (n–2) × ... × 1
▸ 0! = 1 (By definition)
▸ n! = n × (n–1)!

3. Permutations

▸ Number of permutations of n different things taken r at a time:
    P(n, r) = nPr = n! / (n – r)!
▸ If all n things are different and taken all at a time: n!
▸ Permutations of n things with p identical of one kind, q of another:
    = n! / (p! × q! × ...)

4. Circular Permutations

▸ For n distinct objects in a circle: (n – 1)!
▸ If clockwise and anticlockwise arrangements are same: (n – 1)! / 2

5. Combinations

▸ Number of combinations of n different things taken r at a time:
    C(n, r) = nCr = n! / [r! × (n – r)!]
▸ nCr = nC(n – r)
▸ nCr + nC(r – 1) = n+1Cr

6. Relation Between Permutation and Combination

▸ nPr = nCr × r!

Chapter 7: Binomial Theorem

1. Binomial Expansion

▸ (x + a)ⁿ = Σ [nCr × xⁿ⁻ʳ × aʳ], r = 0 to n
▸ Total number of terms = n + 1

2. General Term

▸ Tr+1 = nCr × xⁿ⁻ʳ × aʳ

3. Middle Terms

▸ If n is even, there are 2 middle terms: Tn/2 + 1 and Tn/2
▸ If n is odd, the middle term is T(n+1)/2

4. Properties of Binomial Coefficients

▸ nCr = nC(n – r)
▸ nCr + nC(r – 1) = (n + 1)Cr

Chapter 8: Sequences and Series

1. Arithmetic Progression (A.P.)

▸ General term: an = a + (n – 1)d
▸ Sum of first n terms: Sn = n/2 × [2a + (n – 1)d]
▸ Also: Sn = n/2 × (first term + last term)

2. Geometric Progression (G.P.)

▸ General term: an = arⁿ⁻¹
▸ Sum of first n terms (r ≠ 1): Sn = a × (rⁿ – 1) / (r – 1)
▸ If r < 1: Sn = a × (1 – rⁿ) / (1 – r)

3. Sum to Infinity of G.P. (|r| < 1)

▸ S = a / (1 – r)

4. Arithmetic Mean (A.M.)

▸ A.M. between a and b: (a + b)/2

5. Geometric Mean (G.M.)

▸ G.M. between a and b: √(ab)

6. Special Series

▸ Σn = n(n + 1)/2
▸ Σn² = n(n + 1)(2n + 1)/6
▸ Σn³ = [n(n + 1)/2]²

Chapter 9: Straight Lines

1. General Form

▸ ax + by + c = 0, where a, b, c ∈ ℝ and a, b not both 0

2. Slope of a Line

▸ If line is y = mx + c, then slope = m
▸ Slope between two points (x₁, y₁) and (x₂, y₂): m = (y₂ – y₁) / (x₂ – x₁)

3. Various Forms of Line

▸ Slope-intercept form: y = mx + c
▸ Point-slope form: y – y₁ = m(x – x₁)
▸ Two-point form: (y – y₁)/(x – x₁) = (y₂ – y₁)/(x₂ – x₁)
▸ Intercept form: x/a + y/b = 1
▸ Normal form: xcosα + ysinα = p

4. Angle Between Two Lines

▸ tanθ = |(m₁ – m₂)/(1 + m₁m₂)|

5. Distance from a Point to a Line

▸ D = |ax + by + c| / √(a² + b²)

Chapter 10: Conic Sections

1. Circle

▸ Standard form: (x – h)² + (y – k)² = r²
▸ Centre: (h, k), Radius: r

2. Parabola

▸ Standard: y² = 4ax (opens right), vertex at origin
▸ Focus: (a, 0), Directrix: x = –a, Axis: x-axis
▸ Length of latus rectum: 4a

3. Ellipse

▸ (x²/a²) + (y²/b²) = 1, a > b
▸ Foci: (±ae, 0), where e = √(1 – b²/a²)
▸ Latus rectum: 2b²/a

4. Hyperbola

▸ (x²/a²) – (y²/b²) = 1
▸ Foci: (±ae, 0), where e = √(1 + b²/a²)
▸ Latus rectum: 2b²/a

Chapter 11: Introduction to 3D Geometry

1. Coordinates of a Point

▸ A point P in space is represented by (x, y, z)

2. Distance Formula

▸ Distance between P(x₁, y₁, z₁) and Q(x₂, y₂, z₂):
    = √[(x₂ – x₁)² + (y₂ – y₁)² + (z₂ – z₁)²]

3. Section Formula

▸ If point R divides PQ in ratio m:n, then:
    R = ((mx₂ + nx₁)/(m+n), (my₂ + ny₁)/(m+n), (mz₂ + nz₁)/(m+n))

4. Origin & Axes

▸ X-axis: (x, 0, 0), Y-axis: (0, y, 0), Z-axis: (0, 0, z)
▸ Origin: (0, 0, 0)

Chapter 12: Limits and Derivatives

1. Limit of a Function

▸ limx→a f(x) = L means as x approaches a, f(x) → L
▸ limx→0 sinx/x = 1
▸ limx→0 (1 – cosx)/x² = 1/2

2. Derivative Definition

▸ Derivative of f(x): f'(x) = limh→0 [f(x+h) – f(x)] / h

3. Derivatives of Standard Functions

▸ d/dx (xⁿ) = nxⁿ⁻¹
▸ d/dx (sinx) = cosx
▸ d/dx (cosx) = –sinx
▸ d/dx (eˣ) = eˣ
▸ d/dx (logx) = 1/x

4. Sum and Difference Rule

▸ d/dx [f(x) ± g(x)] = f'(x) ± g'(x)

5. Product and Quotient Rule

▸ Product: d/dx [f(x)·g(x)] = f(x)g'(x) + g(x)f'(x)
▸ Quotient: d/dx [f(x)/g(x)] = [g(x)f'(x) – f(x)g'(x)] / g(x)²

Chapter 13: Statistics

1. Mean (Grouped Data)

▸ Mean (𝑥̄) = Σfixi / Σfi
▸ fi: frequency, xi: class mark

2. Median

▸ Median = l + [(n/2 – F)/f] × h
▸ l = lower boundary of median class
▸ n = total frequency, F = cumulative frequency before median class
▸ f = frequency of median class, h = class width

3. Mode

▸ Mode = l + [(f₁ – f₀) / (2f₁ – f₀ – f₂)] × h
▸ l = lower limit of modal class
▸ f₁ = frequency of modal class
▸ f₀ = frequency before modal class
▸ f₂ = frequency after modal class

Chapter 14: Probability

1. Probability of an Event

▸ P(E) = Number of favourable outcomes / Total number of outcomes
▸ 0 ≤ P(E) ≤ 1

2. Complement of an Event

▸ P(E') = 1 – P(E)

3. Impossible and Sure Events

▸ P(∅) = 0 (Impossible Event)
▸ P(Sample Space) = 1 (Sure Event)
